My Electric Kettle: A User Personas and Storyboarding Task

I wanted to understand the user experience of using a common electric kettle. To gain deep empathy, I observed my roommates when using the kettle and interviewed them. Based this user research, I developed personas and a storyboard

The Electric Kettle

The Kettle is used to boil water/liquid of choice. The lid is lifted by pressing the button on the handle, so that the user can fill it with water. Lid can be closed using the hand to press till it shuts. The ON/OFF button is pressed to initiate heating. Once another click is heard, the water is ready to be poured with handle.

Key Observations

Frequent Users

  • Started with shaking the kettle, then filling kettle with water
  • Did not look at measuring scale
  • Once on, users left to attend other tasks
  • Returned to kettle upon hearing click + when heating sound stopped

First-Time Users

  • First found the top button and pressed it
  • Once kettle lid lifted, user took a 2 second pause before filling with water
  • Also, did not look at measuring scale
  • Moved hand around kettle. Stopped once hand found 'on' button and pressed it

Interview Questions

  • When do you use the kettle?
  • Do you use the kettle regularly? How often do you use the kettle?
  • Can you tell the kettle is working? How can you tell that the that the kettle is working?
  • Do you use the measuring scale?
  • Do you use the measuring scale?
  • On a scale of 1-5 how easy was using the kettle for the first time? (LEARNABILITY)
  • On a scale of 1-5 how easy is it to remember how to use the kettle? (MEMORABILITY)
  • On a scale of 1-5 how efficient is using the kettle? (EFFICIENCY)

Summary of Responses

Usage scenarios

  • Tea
  • Cup Ramen

Usage frequency

  • Twice a week
  • Increases during the winter

Sensorial Feedback from Kettle

  • Blue Light + Click from on button
  • Heating sound to know when the kettle working
  • Heating sound stops + click indicates when heating is complete

Trends & Patterns

  • None of the users used the measuring scale
  • All users said they confused the lid opening button for the turning on button

Conclusion

I found this exercise effective in helping develop empathy for the users of this kettle. I brought my kettle into the the house so I've been using it for a while + I'm used to it. So I was surprised to find that first-time users had problems finding the ON button. It was also interesting to see frequent users optimize their time once they've been using it for a while.
